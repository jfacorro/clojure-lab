Core:
	- Lexer: 
		- Crea los tokens
		- Perece que definiendolos con regex
	- Parser:
		- Analysis sintactico
		- Expresiones validas, es decir, orden de los token que formen expresiones validas
	- Symbol table
		- Parse tree.
	- Compilación o Interpretación:
		- Compilación: genera código.
		- Interpretación: evalúa las expresiones parseadas.
UI:
	- Syntax highlighting:
		- Regex
		- Estrcutura del código: parser, lexer.
